# Misión Lectura

Web app educativa para reforzar comprensión lectora infantil mediante misiones de investigación, puntos, medallas y ranking positivo entre compañeros.

## Incluye

- Inicio gamificado de Misión Lectura
- Selección de nivel escolar
- Dashboard del estudiante
- Mapa de misiones
- Historia corta con palabras clave
- Retos de comprensión lectora
- Resultado con puntos y medallas
- Ranking de compañeros
- Panel simple para maestro

## Ejecutar localmente

```bash
npm install
npm run dev
```

Luego abre:

```text
http://localhost:3000
```

## Despliegue recomendado

Este proyecto está listo para desplegarse en Vercel como una app Next.js.
